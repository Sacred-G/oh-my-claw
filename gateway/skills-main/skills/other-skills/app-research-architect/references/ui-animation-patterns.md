# UI and Animation Patterns

## Visual Direction

Use shadcn/ui and Tailwind as a clean base, then add a distinct product identity through spacing, typography, color system, motion, icons, and custom hero/dashboard compositions.

## Motion Stack

- Motion/Framer Motion: React component transitions, layout animations, page transitions, gestures.
- GSAP: timeline-heavy sequences, scroll animation, SVG/path morphing, advanced choreography.
- Three.js/React Three Fiber: 3D scenes, particles, product visualizations, spatial interfaces.
- Remotion: generate videos, previews, explainers, exportable media from React components.

## Catmull-Rom Uses

- Smooth 3D camera flythroughs.
- Animated particles/objects moving through product workflows.
- Curved graph edges, automations, timelines, route paths.
- Scroll-linked hero animations with smooth spline interpolation.

## Professional UI Patterns

- Command palette and keyboard-first workflows.
- Split-pane workspaces with inspector panels.
- Timeline/activity feed for AI/tool actions.
- Canvas/node graph for agent workflows or automation builders.
- Rich empty states with templates and quick actions.
- Realtime presence indicators for collaborative apps.
- Skeleton loading, optimistic updates, toast/action undo.
- Polished settings: profile, workspace, billing, API keys, webhooks.

## Guardrails

- Keep animation purposeful: explain state, guide attention, reward completion.
- Respect accessibility: reduced-motion support, keyboard navigation, contrast.
- Avoid overloading dashboards with 3D unless it helps comprehension or brand.
